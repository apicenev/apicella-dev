apicella.dev – Developer Portfolio

A personal one-page portfolio built with static HTML and Tailwind CSS, designed to clearly present who I am as a software engineer, the technologies I work with, and the projects I’m building.

The site is structured with reusable HTML partials, a small JavaScript include helper, and a lightweight Tailwind setup compiled via npm scripts. It’s deployed on Vercel and connected to GitHub, giving me a simple but professional workflow for iterating on the design and content.

BookTrackr – Reading & Knowledge Hub

BookTrackr is a personal web application that transforms reading into a structured learning process. Instead of only tracking which books are finished, it focuses on what I’ve learned, how I apply those ideas in practice, and how my knowledge grows over time.

The app lets me manage my reading pipeline (wishlist → currently reading → finished), document key insights in a structured way, and link each book to real projects (for example GitHub repositories) where I applied the concepts. A simple recommender will help decide what to read next based on interests and previous books.
